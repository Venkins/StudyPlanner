# StudyPlanner - 智能学习计划工具

基于 HTML + JavaScript 的多周期学习计划管理工具，支持日/周/月/季度四种时间维度，提供可视化时间轴和多种导出方式。

![CSS3](https://img.shields.io/badge/CSS3-1572B6?logo=css3&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?logo=javascript&logoColor=black)

## 功能特性

- **📅 多周期规划** - 支持日计划、周计划、月计划、季度计划四种视图切换
- **⏰ 时间块管理** - 为任务分配精确的开始/结束时间段
- **🏷️ 任务分类** - 内置学习/锻炼/休息/娱乐/工作/饮食/其他七大分类
- **🎨 颜色标记** - 10 种颜色可选，可视化区分不同任务
- **🔄 重复规则** - 支持仅当天、每天、工作日、周末、每周五种重复模式
- **📊 可视化时间轴** - 按天展示时间块分布，直观查看日程安排
- **📋 列表视图** - 以列表形式查看所有任务详情
- **📈 统计面板** - 实时显示任务数、已规划时长、空闲时间、分类数量
- **🔄 批量操作** - 复制到本周全部天数、复制到下周、清空当前周期
- **📤 多种导出** - 支持 JSON、CSV、打印/PDF 导出
- **💾 本地存储** - 基于 localStorage 自动保存，刷新不丢失
- **📱 响应式布局** - 适配桌面和移动设备

## 使用方式

无需构建，直接打开 `学习计划工具.html` 即可使用：

```bash
# 或用本地服务器
npx serve .
# 然后在浏览器中打开 学习计划工具.html
```

```
StudyPlanner/

```

## 操作说明

1. **选择周期** - 顶部切换日/周/月/季度视图
2. **添加任务** - 左侧面板填写时间、名称、分类、颜色和重复规则后点击添加
3. **编辑/删除** - 列表视图中点击编辑图标修改任务，点击删除图标移除
4. **批量复制** - 在周视图中可将当天任务复制到本周或下周
5. **导出** - 右上角导出按钮支持 JSON、CSV、打印/PDF

## 技术要点

- 纯前端实现，无后端依赖
- CSS 变量统一管理主题色
- localStorage 持久化存储
- 支持打印样式（`@media print`）
- 响应式设计（`@media max-width: 900px`）
